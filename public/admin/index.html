📥 Bulk Import Questions

Paste a JSON array of questions. Each question needs at least: "question", "type" (single/multi/numerical), "answer". Optional: "options" (array), "sectionId" (number), "subsection" (string), "images" (array of {id, src, alt}), "tables" (array of {id, caption, headers, rows}).

Manage Sections

Drag sections to reorder. Deleting a section removes questions from that section (sets them to unassigned). You can add, edit label/type, reorder, or remove subsections within each section. Questions reference subsections by ID; if you remove a subsection, related questions become unassigned from that subsection.
